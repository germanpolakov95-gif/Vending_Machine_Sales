# Vending Machine Sales Analysis
Excel pet project for the analysis of sales of vending machines

In this project, I analyzed data from Kaggle on vending machine sales in New Jersey.

**What was analyzed**: Product analysis, Location Rating by total revenue, Revenue dynamics by month, and the ratio of card and cash purchases.

Link to the dataset: [Vending Machine Sales](https://www.kaggle.com/datasets/awesomeasingh/vending-machine-sales)

## About the repository
 - dashboard.xlsx (final project with visualizations)
 - dashboard_screenshot.png (to view the result quickly)
 - vending_machine_sales.csv (source dataset)

In the dashboard.xlsx you will see a sheet with the final visualizations.  You can also view hidden worksheets if you wish.
